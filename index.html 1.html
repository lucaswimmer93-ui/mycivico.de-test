Häufige Fragen

FAQ

Was kostet Civico für Freiwillige? +
Civico ist für Freiwillige vollständig kostenlos. Immer.
Kann ich Civico erst testen? +
Ja. Civico kann zunächst im kleinen Rahmen getestet werden, bevor ein dauerhaftes Modell startet.
In welchen Sprachen ist die App verfügbar? +
Civico ist auf Deutsch, Englisch, Türkisch, Russisch und Polnisch verfügbar.
Wie werden meine Daten geschützt? +
Alle Daten werden auf EU-Servern gespeichert (Supabase Frankfurt). Civico ist vollständig DSGVO-konform.
Gibt es eine mobile App? +
Civico ist eine Progressive Web App (PWA) – sie funktioniert auf jedem Gerät ohne Download und kann zum Homescreen hinzugefügt werden.
